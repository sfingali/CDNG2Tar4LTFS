# CDNG2Tar4LTFS (quick start)

This project packages CDNG sequence directories into LTFS-friendly TAR files and generates two screenshot previews per sequence.

## Install (pip)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python cdng_tar_gui.py
```

For full usage details and LTFS notes, see `README.md`.
